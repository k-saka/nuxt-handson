# nuxt-handson

- Nuxt.jsに入門するためのハンズオンです。
- シンプルなWebアプリケーションを段階的に実装していきます。
- 特定のツールを使用せずに、ゼロからNuxt.jsをインストールしてプロジェクトを作成します。
- Nuxt.jsの標準機能のほか、API通信にaxios、UIコンポーネントにVuetify.jsを使用しています。
